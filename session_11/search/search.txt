Web API

	Web APIs هي واجهات توفرها المتصفح تتيح لجافاسكريبت التفاعل مع المتصفح والبيئة الخارجية لمحرك جافاسكريبت. أمثلة على ذلك:
	DOM (نموذج كائن المستند): للتعامل مع HTML و CSS.
	Fetch API: لإجراء طلبات الشبكة.
	setTimeout و setInterval: لأحداث التوقيت.
	Geolocation API: للوصول إلى موقع المستخدم

Event Loop

	Event Loop هو مفهوم أساسي في جافاسكريبت يتعامل مع العمليات غير المتزامنة. إليك شرح مبسط:
	Call Stack: حيث يتم تنفيذ الدوال. تُضاف الدوال إلى المكدس عند استدعائها وتُزال عند إرجاعها.
	Web APIs: عندما يتم استدعاء عملية غير متزامنة (مثل setTimeout)، يتم التعامل معها بواسطة Web API، التي تنتظر حتى تكتمل العملية.
	Callback Queue: بمجرد أن تكمل Web API العملية، تدفع الدالة الاسترجاعية إلى قائمة الانتظار.
	Event Loop: يتحقق باستمرار من Call Stack. إذا كان Call Stack فارغًا، يأخذ أول دالة من Callback Queue ويدفعها إلى Call Stack للتنفيذ.
